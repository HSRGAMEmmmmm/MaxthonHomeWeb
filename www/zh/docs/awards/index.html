傲游奖项

傲游 - 您的互联网起点

傲游是一款最先进的多平台网页浏览器，经常超越其他顶级浏览器，为用户提供无缝的浏览和共享体验。
感谢我们的支持者，傲游已成为全球现象：一个由朋友、同事和粉丝组成的社区，他们相信上网应该是快速和有趣的。

傲游的历史与荣誉

  • 2014年 - 傲游在About.com读者选择奖的三个类别中占据主导地位。
  • 2013年 - 傲游在About.com读者选择奖中赢得三个第一名奖项。
  • 2012年 - 傲游发布傲游云浏览器，为iOS、Windows、Android和Mac提供多设备无缝浏览。
  • 2012年 - 傲游推出浏览器的第三方隐私政策，可以隐藏浏览中生成的cookie，并阻止网站通过cookie追踪用户隐私。
  • 2012年 - 傲游发布Windows版傲游云浏览器，提供云标签、云下载和云推送等创新云功能。
  • 2011年 - 傲游获得CNET的五星奖，并被评为最佳免费软件。
  • 2011年 - 傲游与Fuhu和Kingston建立新的分销合作伙伴关系。
  • 2011年 - 傲游发布Android平板浏览器。
  • 2011年 - 傲游被PCWorld评为100个最佳产品之一。
  • 2010年 - 傲游3集成了更多的安全功能，包括支持“请勿追踪”协议、用户帐户数据加密、安全网站验证和危险网站屏蔽。
  • 2010年 - 傲游3新增阅读模式和夜间模式，为用户提供更健康的浏览体验。
  • 2010年 - 傲游3开始支持Webkit核心和Trident核心浏览。
  • 2010年 - 傲游为Android重新定义触屏移动浏览。傲游浏览器下载量达到5亿次。
  • 2009年 - 傲游3桌面版首次亮相，并被选中出现在欧洲“浏览器选择”屏幕上。
  • 2008-2009年 - 傲游赢得CNET Webware 100奖项。
  • 2007年 - 傲游2 for Windows发布，提供收藏夹同步、广告拦截、资源嗅探器、分屏模式等多项优秀功能。
  • 2007年 - 傲游成为第一个提供沙盒标签以增强安全性的浏览器。傲游赢得第二个Webware 100奖项，击败所有其他浏览器。
  • 2006年 - 傲游与谷歌、微软、百度和Yandex合作。
  • 2005年 - 傲游获得Red Herring亚洲100强奖。
  • 2005年 - 傲游获得ListSoft用户选择奖。
  • 2005年 - 傲游成为第一个提供云服务以同步书签和历史记录的浏览器。
  • 2004年 - MyIE2成为傲游，扩展了公众对创新浏览器真正威力的期望。
  • 2003年 - MyIE2重新定义了浏览。
