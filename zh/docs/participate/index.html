您有多种方式参与傲游

全世界的傲游社区成员正与傲游团队一起努力，使我们的产品更加完善。如果您对帮助傲游（并了解一些有趣的软件和产品开发）感兴趣，下面列出了您可以参与的领域。请阅读描述，看看哪个最适合您：

加入我们的全球翻译团队

傲游是一款在全球广泛使用的产品，因此需要不断改进翻译和本地化。我们真诚邀请您成为我们的官方翻译或校对人员，为傲游浏览器做翻译或校对工作。

担任全球版主，引导傲游论坛的讨论

论坛 是傲游的官方讨论社区，我们现在正在寻找具有出色沟通技巧的版主，他们在社区中活跃，并且有帮助他人的热情。作为版主，您在论坛的成功中扮演着关键角色，通过引导和指导讨论来促进社区发展。

成为傲游官方博客的贡献编辑

傲游博客 是增强沟通、传播傲游文化、品牌推广或其他公共关系目的的官方渠道。我们欢迎您成为 傲游技巧 的贡献编辑。您可以分享您的使用经验，也可以谈谈您自己的独特见解。

成为傲游大使

大使是我们总体本地化计划的一部分。您应该与傲游员工（包括 Jeff）密切合作，帮助在您的国家推广傲游。您可以组织一些线上或线下活动，并充当您所在国家所有傲游用户的首席代表。

为傲游开发扩展

为了改进我们的产品，许多热情的用户正在利用他们的时间和技能来帮助开发傲游扩展。您可以成为这些扩展开发者之一，并在我们的 扩展中心 中拥有一个以您的名字命名的插件。

为傲游浏览器设计多种皮肤或主题

正如您在 MX4 的设置中所看到的，我们有由亲爱的傲游用户编写的彩色皮肤和外观主题选项。现在我们也诚挚地邀请您成为 MX5 的皮肤设计师。

对您的好处

您可以以上述任何方式帮助塑造傲游，您也可能需要投入一些时间和精力来完成它。然而，您的努力将永远得到回报。

您将获得许多终身受益的个人好处：

  • 接受挑战
  • 获得经验
  • 学习新技能
  • 做出改变
  • 通过实现目标获得信心
  • 动力和成就感
  • 您作品的作者身份
  • 在我们官方网站上获得正式认可
  • 成为傲游的一部分
  • 培养新的兴趣和爱好
  • 潜在的工作机会和提升职业选择
  • 服务于您所在国家甚至全球的所有用户
  • 与志同道合的朋友或其他傲游用户建立新友谊
  • 享受乐趣！

招募规则

感谢您对志愿为傲游贡献时间的兴趣！以下是每个角色的一些期望：

译者

  • 目标语言的母语水平理解
  • 扎实的英语语言（源语言）理解
  • 可靠、负责和足智多谋

版主

  • 在论坛活跃超过半年
  • 每天有至少2小时的时间进行论坛管理
  • 与傲游的内部团队密切合作，以改进论坛并解决用户问题
  • 在社区帖子中分享相关和有用的信息。通过合并、移动、指导帖子来引导和组织论坛
  • 维护傲游的品牌和形象，以傲游员工的身份进行管理

贡献编辑

  • 擅长理解和表达英语作为第一语言
  • 对傲游和其他相关领域有极大的热情

大使

  • 对傲游和新技术充满热情
  • 活跃的傲游用户
  • 参与傲游的社区和活动
  • 帮助不同地区的产品本地化
  • 获得傲游授权后监控当地社交媒体平台

扩展开发者

  • 对傲游和其他主要网络浏览器（例如 Chrome、IE...）感兴趣并熟悉
  • 擅长 JavaScript、HTML 和 CSS
  • 有插件开发经验者优先

皮肤设计师

  • 具备出色的视觉设计技能，精通 Photoshop、Illustrator 或其他线框工具
  • 对傲游的皮肤和主题设计感兴趣
  • 有 UI 设计经验者优先
